# Examples Directory

This directory contains organized examples and demonstrations for the OpenCode project.

## Structure

- `demos/` - General demonstration scripts
- `model-demos/` - AI model integration examples
- `integration-demos/` - System integration demonstrations
- `scripts/` - Utility scripts and examples

## Usage

These examples showcase various features and integrations. They are organized by category for easy navigation.

## Contributing

When adding new examples:

1. Choose the appropriate subdirectory
2. Include clear documentation
3. Test the example works correctly
4. Update this README if needed
